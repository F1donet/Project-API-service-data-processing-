# Project-API-service-data-processing-
Проект от компании "E-soft": API-сервис для обработки данных на Flask
API позволяет загружать файлы (CSV, Excel) и сохраняет их в PostgreSQL.
Реализован анализ данных с помощью Pandas (среднее, медиана, корреляция).
Есть функционал очистки данных (удаление дубликатов, заполнение пропущенных значений).
API корректно обрабатывает ошибки загрузки файлов (400 Bad Request, 415 Unsupported Media Type).
Реализованы следующие эндпоинты:
POST /upload для загрузки файла
GET /data/stats для получения аналитики по загруженным данным
GET /data/clean для очистки данных
(Продвинутый уровень) GET /data/plot для получения графика
